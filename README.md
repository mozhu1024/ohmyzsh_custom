# ohmyzsh_custom

Modify **.zshrc**

```shell

# Set name of the theme to load
ZSH_THEME="mozhu"

# custom folder than $ZSH/custom
ZSH_CUSTOM=/Users/yourname/path/ohmyzsh_custom

# Standard plugins
plugins=(
    git
    gitignore
    autojump
    vscode
    1password
    # Custom plugins
    zsh-autosuggestions
    zsh-syntax-highlighting
    mozhu
)
```
